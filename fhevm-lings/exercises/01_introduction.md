**FHElings** is an interactive learning tool inspired by Rustlings, designed to teach you **FHEVM (Fully Homomorphic Encryption Virtual Machine)** through hands-on practice.

## What is FHEVM?

FHEVM is a revolutionary technology developed by **Zama** that allows you to perform computations on encrypted data without ever decrypting it. This enables:

-  **Private smart contracts** - Keep data encrypted on-chain
-  **Confidential gaming** - Hidden information (fog of war) on blockchain
-  **Secret voting** - Truly private on-chain governance
-  **Sealed auctions** - Bid without revealing your amount
-  **Privacy-preserving analytics** - Compute on sensitive data

## How FHElings Works

1. **Read the exercise** - Each exercise has a TODO comment
2. **Write the solution** - Fill in the missing code
3. **Save the file** - The system automatically tests your solution
4. **Get feedback** - Instant hints if something is wrong
5. **Advance** - Press 'n' to go to the next exercise
