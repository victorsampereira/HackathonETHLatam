# ⚡ fhevm-lings

Bem-vindo ao `fhevm-lings`! Um conjunto de pequenos exercícios interativos para aprender a usar o FHEVM da Zama.

O conceito é simples: executar a ferramenta, corrigir o erro e gravar o ficheiro para avançar para o próximo desafio.

## 1. Instalação

1.  **Clone o projeto** (ou crie os ficheiros localmente):
    ```bash
    git clone [https://github.com/SEU_USER/fhevm-lings.git](https://github.com/SEU_USER/fhevm-lings.git)
    cd fhevm-lings
    ```
    *(Nota: Terá de criar este repositório no GitHub depois de adicionar os ficheiros.)*

2.  **Instale as dependências** (vai precisar do Node.js v18+):
    ```bash
    npm install
    ```

## 2. Como Usar

Basta executar o comando "watch":

```bash
npm run watch
